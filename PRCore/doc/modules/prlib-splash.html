prlib.showSplashScreen ([Table])
Show Splash Screen. dispay a full screen splash using specified options

Parameters:

  • Table obj containing options to define behaveor of the splash (optional)

Tables

obj
Table to be passed to

Fields:

  • image string Path to image in resources directory. Defaults to the Probably Rational logo. (optional)
  • effect string Effect that the image will have. Values can be "static", or "zoom". (default 'static')
  • background Table value that represents color. This is takes any standard corona color format of `{ gray }`, `{ gray, alpha }`, `{ red, green, blue }`, or `{ red, green, blue, alpha }`. (default {0})
  • time int Time taken in milliseconds for full animation to complete. (default 5000)
  • onComplete Function to be called after animation ends. (optional)
